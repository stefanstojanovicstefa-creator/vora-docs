# Thai

<Tabs>
  <Tab title="Simple">
    Vora fully supports **Thai** for voice AI agents. ~60M speakers in Thailand.

    ### Setting Up a Thai Agent

    <Steps>
      <Step title="Create your agent">
        Use any creation method (Interview, URL Import, etc.)
      </Step>
      <Step title="Set language to Thai">
        In voice settings, select **Thai** from the language dropdown.
      </Step>
      <Step title="Choose a voice">
        Select from available Thai voices in the voice library.
      </Step>
      <Step title="Test and deploy">
        Test with the simulator, then deploy as usual.
      </Step>
    </Steps>

    <Tip>
      For best results with Thai, we recommend Murf AI voices for the most natural speech.
    </Tip>
  </Tab>
  <Tab title="Advanced">
    ## Provider Matrix

    | Detail | Value |
    |--------|-------|
    | **Code** | th-TH |
    | **Region** | Thailand |
    | **Speakers** | ~60M |
    | **Best ASR** | Valsea |
    | **Best TTS** | Murf AI |
    | **Best LLM** | Qwen |

    ### Available Providers

    | Type | Providers |
    |------|-----------|
    | **ASR** | Valsea (SEA-optimized), Google, Azure, AWS, Deepgram, Soniox, AssemblyAI, ElevenLabs, Speechmatics, iFLYTEK |
    | **TTS** | Google, Azure, AWS Polly, ElevenLabs, Murf AI, Speechify |
    | **LLM** | Qwen (Best), GPT-4o, Gemini, Claude |

    ### Recommended Stack

    For the best Thai voice experience:
    - **STT**: Valsea (best accuracy for Thai)
    - **LLM**: Qwen (best Thai understanding)
    - **TTS**: Murf AI (most natural Thai speech)

    ### Configuration

    In the Command Center:
    1. **Ears Tab** → Set STT provider to Valsea, language to `th-TH`
    2. **Brain Tab** → Set LLM to Qwen
    3. **Voice Tab** → Set TTS to Murf AI, select a Thai voice
  </Tab>
</Tabs>
