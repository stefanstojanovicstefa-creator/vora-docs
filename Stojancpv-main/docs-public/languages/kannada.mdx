# Kannada

<Tabs>
  <Tab title="Simple">
    Vora fully supports **Kannada** for voice AI agents. ~45M speakers in India (Karnataka).

    ### Setting Up a Kannada Agent

    <Steps>
      <Step title="Create your agent">
        Use any creation method (Interview, URL Import, etc.)
      </Step>
      <Step title="Set language to Kannada">
        In voice settings, select **Kannada** from the language dropdown.
      </Step>
      <Step title="Choose a voice">
        Select from available Kannada voices in the voice library.
      </Step>
      <Step title="Test and deploy">
        Test with the simulator, then deploy as usual.
      </Step>
    </Steps>

    <Tip>
      For best results with Kannada, we recommend Sarvam AI voices for the most natural speech.
    </Tip>
  </Tab>
  <Tab title="Advanced">
    ## Provider Matrix

    | Detail | Value |
    |--------|-------|
    | **Code** | kn-IN |
    | **Region** | India (Karnataka) |
    | **Speakers** | ~45M |
    | **Best ASR** | Sarvam AI |
    | **Best TTS** | Sarvam AI |
    | **Best LLM** | Sarvam 2B |

    ### Available Providers

    | Type | Providers |
    |------|-----------|
    | **ASR** | Sarvam AI (Best), Google, Azure, AWS, Deepgram, Soniox |
    | **TTS** | Sarvam AI (Best), Google, Azure, Murf AI |
    | **LLM** | Sarvam 2B (Best), GPT-4o, Gemini, Claude |

    ### Recommended Stack

    For the best Kannada voice experience:
    - **STT**: Sarvam AI (best accuracy for Kannada)
    - **LLM**: Sarvam 2B (best Kannada understanding)
    - **TTS**: Sarvam AI (most natural Kannada speech)

    ### Configuration

    In the Command Center:
    1. **Ears Tab** → Set STT provider to Sarvam AI, language to `kn-IN`
    2. **Brain Tab** → Set LLM to Sarvam 2B
    3. **Voice Tab** → Set TTS to Sarvam AI, select a Kannada voice
  </Tab>
</Tabs>
