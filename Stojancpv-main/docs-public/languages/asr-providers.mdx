## Key Differences

### Latency

Real-time voice agents need low-latency STT. Rankings:
1. **Deepgram** - Lowest latency (~100ms)
2. **Soniox** - Very low (~150ms)
3. **Google/Azure** - Low (~200-300ms)
4. **AssemblyAI** - Moderate (~300ms)

### Accuracy

Accuracy varies by language and accent:
- **English**: Deepgram and AssemblyAI lead
- **Indian languages**: Sarvam AI leads
- **Arabic**: Munsit leads (25+ dialect support)
- **SEA languages**: Valsea leads (Singlish, accented English)

### Pricing

Most cost-effective options:
1. **Deepgram** - $0.0077/min (best value for quality)
2. **Soniox** - $0.10/hr async ($0.0017/min)
3. **Google** - $0.016/min
4. **Azure** - $0.017/min

## Configuration

Configure ASR providers in the Command Center [Ears Tab](/advanced/command-center/ears-tab):
1. Select your primary STT provider
2. Set the language code
3. Configure VAD (voice activity detection) settings
4. Optionally set a fallback provider in the [Advanced Tab](/advanced/command-center/advanced-tab)
