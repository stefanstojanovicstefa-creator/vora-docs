# Malay

<Tabs>
  <Tab title="Simple">
    Vora fully supports **Malay** for voice AI agents. ~80M speakers in Malaysia, Singapore, and Brunei.

    ### Setting Up a Malay Agent

    <Steps>
      <Step title="Create your agent">
        Use any creation method (Interview, URL Import, etc.)
      </Step>
      <Step title="Set language to Malay">
        In voice settings, select **Malay** from the language dropdown.
      </Step>
      <Step title="Choose a voice">
        Select from available Malay voices in the voice library.
      </Step>
      <Step title="Test and deploy">
        Test with the simulator, then deploy as usual.
      </Step>
    </Steps>

    <Tip>
      For best results with Malay, we recommend Murf AI voices for the most natural speech.
    </Tip>
  </Tab>
  <Tab title="Advanced">
    ## Provider Matrix

    | Detail | Value |
    |--------|-------|
    | **Code** | ms-MY |
    | **Region** | Malaysia / Singapore / Brunei |
    | **Speakers** | ~80M |
    | **Best ASR** | Valsea / Deepgram |
    | **Best TTS** | Murf AI |
    | **Best LLM** | Qwen |

    ### Available Providers

    | Type | Providers |
    |------|-----------|
    | **ASR** | Valsea (SEA-optimized), Google, Azure, AWS, Deepgram, Soniox, AssemblyAI, ElevenLabs, Speechmatics, iFLYTEK |
    | **TTS** | Google, Azure, AWS Polly, ElevenLabs, Murf AI, Speechify |
    | **LLM** | Qwen (Best), GPT-4o, Gemini, Claude, Cohere Aya |

    ### Recommended Stack

    For the best Malay voice experience:
    - **STT**: Valsea / Deepgram (best accuracy for Malay)
    - **LLM**: Qwen (best Malay understanding)
    - **TTS**: Murf AI (most natural Malay speech)

    ### Configuration

    In the Command Center:
    1. **Ears Tab** → Set STT provider to Valsea or Deepgram, language to `ms-MY`
    2. **Brain Tab** → Set LLM to Qwen
    3. **Voice Tab** → Set TTS to Murf AI, select a Malay voice
  </Tab>
</Tabs>
