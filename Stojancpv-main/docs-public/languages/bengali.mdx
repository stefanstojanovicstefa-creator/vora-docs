# Bengali

<Tabs>
  <Tab title="Simple">
    Vora fully supports **Bengali** for voice AI agents. ~230M speakers in India and Bangladesh.

    ### Setting Up a Bengali Agent

    <Steps>
      <Step title="Create your agent">
        Use any creation method (Interview, URL Import, etc.)
      </Step>
      <Step title="Set language to Bengali">
        In voice settings, select **Bengali** from the language dropdown.
      </Step>
      <Step title="Choose a voice">
        Select from available Bengali voices in the voice library.
      </Step>
      <Step title="Test and deploy">
        Test with the simulator, then deploy as usual.
      </Step>
    </Steps>

    <Tip>
      For best results with Bengali, we recommend Sarvam AI voices for the most natural speech.
    </Tip>
  </Tab>
  <Tab title="Advanced">
    ## Provider Matrix

    | Detail | Value |
    |--------|-------|
    | **Code** | bn-IN |
    | **Region** | India and Bangladesh |
    | **Speakers** | ~230M |
    | **Best ASR** | Sarvam AI |
    | **Best TTS** | Sarvam AI |
    | **Best LLM** | Sarvam 2B |

    ### Available Providers

    | Type | Providers |
    |------|-----------|
    | **ASR** | Sarvam AI (Best), Google, Azure, AWS, Deepgram, Soniox, ElevenLabs, Speechmatics, iFLYTEK |
    | **TTS** | Sarvam AI (Best), Google, Azure, AWS Polly, ElevenLabs, Murf AI, Speechify |
    | **LLM** | Sarvam 2B (Best), GPT-4o, Gemini, Claude, Cohere Aya |

    ### Recommended Stack

    For the best Bengali voice experience:
    - **STT**: Sarvam AI (best accuracy for Bengali)
    - **LLM**: Sarvam 2B (best Bengali understanding)
    - **TTS**: Sarvam AI (most natural Bengali speech)

    ### Configuration

    In the Command Center:
    1. **Ears Tab** → Set STT provider to Sarvam AI, language to `bn-IN`
    2. **Brain Tab** → Set LLM to Sarvam 2B
    3. **Voice Tab** → Set TTS to Sarvam AI, select a Bengali voice
  </Tab>
