# Language Support

Vora supports 15+ languages using combinations of Deepgram, Google, AssemblyAI, ElevenLabs, and Gemini. Each language page shows which providers work best.

    ### Set Your Agent's Language

    <Steps>
      <Step title="Open your agent">
        Go to **Agents** and select your agent.
      </Step>
      <Step title="Choose a language">
        In the voice settings, select the language from the dropdown.
      </Step>
      <Step title="Save">
        Click **Save**. Your agent now understands and responds in the selected language.
      </Step>
    </Steps>

    ## Configuration

    Set language per-agent in the Command Center:
    - **Ears Tab** — STT language for speech recognition
    - **Voice Tab** — TTS language and voice selection
    - **Brain Tab** — LLM system prompt language context

    For optimal results, verify that all three components support your target language before deploying.
