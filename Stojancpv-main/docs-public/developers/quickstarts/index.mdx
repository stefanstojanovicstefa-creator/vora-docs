# Developer Quickstarts

Choose your preferred language or tool to get started with the Vora API. Each quickstart walks you through creating your first voice agent and starting a conversation.

## What You'll Build

In each quickstart, you'll:

<Steps>
  <Step title="Set Up Authentication">
    Configure your API key and initialize the SDK
  </Step>
  <Step title="Create an Agent">
    Define a voice agent with a system prompt and voice settings
  </Step>
  <Step title="Start a Session">
    Connect to a voice session and enable audio
  </Step>
  <Step title="Have a Conversation">
    Talk to your agent and see transcripts in real-time
  </Step>
</Steps>

## Sample Code Repository

Find complete examples in our GitHub repository:

```bash
git clone https://github.com/vora-ai/examples
cd examples
```

The repository includes:

- **Basic examples** - Minimal code to get started
- **Full applications** - Production-ready templates
- **Integration examples** - CRM, calendar, e-commerce integrations
- **Framework starters** - Next.js, Express, FastAPI templates
