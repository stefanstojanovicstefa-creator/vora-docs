## Configuration

### Basic Setup

```typescript
const agent = await vora.agents.create({
  name: 'EU Agent',
  systemPrompt: 'You are a helpful assistant.',
  model: {
    provider: 'mistral',
    model: 'mistral-large-latest'
  }
});
```

### Advanced Options

```typescript
const agent = await vora.agents.create({
  name: 'Custom Agent',
  systemPrompt: 'You are a professional assistant.',
  model: {
    provider: 'mistral',
    model: 'mistral-large-latest',
    temperature: 0.7,
    maxTokens: 1024,
    topP: 0.9,
    safePrompt: true  // Enable safe mode
  }
});
```

## Why Mistral?

### 1. EU Data Residency

Mistral is headquartered in Paris and processes data in the EU:
- GDPR compliant by default
- Data doesn't leave the EU
- Ideal for European enterprises

### 2. Open-Source Heritage

Benefits of open-source foundation:
- Transparent model behavior
- No lock-in concerns
- Community-vetted quality

### 3. Competitive Quality

Mistral Large competes with GPT-4:
- Strong reasoning capabilities
- Good instruction following
- Excellent multilingual support

### 4. European Language Support

Strong performance in:
- French, German, Spanish, Italian
- Other European languages
- Regional dialects

## Function Calling

Mistral has robust function calling support:

```typescript
const agent = await vora.agents.create({
  name: 'Tool Agent',
  systemPrompt: 'You help with various tasks.',
  model: {
    provider: 'mistral',
    model: 'mistral-large-latest'
  },
  functions: [
    {
      name: 'search_products',
      description: 'Search for products in the catalog',
      parameters: {
        type: 'object',
        properties: {
          query: { type: 'string' },
          category: { type: 'string' },
          maxPrice: { type: 'number' }
        },
        required: ['query']
      },
      handler: {
        type: 'webhook',
        url: 'https://api.example.com/products/search'
      }
    }
  ]
});
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Rate Limits">
    Mistral rate limits vary by tier:

    - Check current limits in Mistral Console
    - Request increases for production use
    - Configure fallback provider
  </Accordion>

  <Accordion title="EU vs US Latency">
    If calling from outside EU:

    - Expect ~50-100ms additional latency
    - Consider using EU-based Vora deployment
    - Use fallback for latency-critical paths
  </Accordion>

  <Accordion title="Context Length">
    If hitting context limits:

    - Large: 128K tokens
    - Medium/Small: 32K tokens
    - Use summarization for long conversations
  </Accordion>
</AccordionGroup>
