# Deepgram

Deepgram's Nova-3 model offers the best combination of speed and accuracy for real-time voice applications. **Deepgram is Vora's default STT provider.**

## Configuration

### Basic Setup

```typescript
const agent = await vora.agents.create({
  name: 'Voice Agent',
  systemPrompt: 'You are a helpful assistant.',
  stt: {
    provider: 'deepgram',
    model: 'nova-3'
  }
});
```

### Full Configuration

```typescript
const agent = await vora.agents.create({
  name: 'Custom Agent',
  systemPrompt: 'You are a specialized assistant.',
  stt: {
    provider: 'deepgram',
    model: 'nova-3',
    language: 'en-US',
    punctuate: true,
    numerals: true,
    profanityFilter: false,
    keywords: ['your-product', 'specific-terms'],
    interimResults: true,
    utteranceEndMs: 800,
    endpointing: 300
  }
});
```

---

## Configuration Options

### Core Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `model` | string | `nova-3` | STT model |
| `language` | string | `en` | Language code |
| `punctuate` | boolean | `true` | Add punctuation |
| `numerals` | boolean | `false` | Convert numbers |

### Real-Time Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `interimResults` | boolean | `true` | Partial transcripts |
| `utteranceEndMs` | number | `1000` | Silence before end |
| `endpointing` | number | `300` | Sentence detection (ms) |

### Accuracy Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `keywords` | string[] | `[]` | Boost specific words |
| `profanityFilter` | boolean | `false` | Filter profanity |
| `redact` | string[] | `[]` | Redact PII types |

## Best Practices

<AccordionGroup>
  <Accordion title="Optimize Latency">
    ```typescript
    stt: {
      provider: 'deepgram',
      model: 'nova-3',
      endpointing: 200,
    ```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Poor Accuracy">
    Add keywords for domain terms, check audio quality, try `nova-3-phonecall` for phone audio, ensure correct language code.
  </Accordion>

  <Accordion title="High Latency">
    Lower `utteranceEndMs`, reduce `endpointing` threshold, check network latency, enable interim results.
  </Accordion>

  <Accordion title="Cutting Off Speech">
    Increase `utteranceEndMs` (try 1200ms) and `endpointing` (try 500ms).
  </Accordion>
</AccordionGroup>
