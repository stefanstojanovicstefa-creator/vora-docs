
```typescript
const agent = await vora.agents.create({

    provider: 'elevenlabs',

    similarityBoost: 0.75,
    style: 0.0,

```

## Voice Settings

### Stability

Controls voice consistency vs expressiveness:

```typescript
stability: 0.5  // 0.0 - 1.0
```

| Value | Effect |
|-------|--------|
| 0.0 - 0.3 | Very expressive, varied |
| 0.3 - 0.5 | Balanced |
| 0.5 - 0.7 | Consistent |
| 0.7 - 1.0 | Very stable, monotone |

### Similarity Boost

How closely to match the original voice:

```typescript
similarityBoost: 0.75  // 0.0 - 1.0
```

| Value | Effect |
|-------|--------|
| 0.0 - 0.3 | More variation |
| 0.5 | Balanced |
| 0.7 - 1.0 | Very close to original |

### Style

Exaggeration of voice style:

```typescript
style: 0.0  // 0.0 - 1.0
```

<Warning>
Higher style values increase latency. Keep at 0 for real-time voice AI.
</Warning>

### Speaker Boost

Enhance voice clarity:

```typescript
useSpeakerBoost: true
```

Recommended for phone/noisy environments.

## Voice Cloning

### Instant Voice Clone

Create a voice from a sample:

```typescript
// Through ElevenLabs API
const voice = await elevenlabs.voices.add({
  name: 'My Custom Voice',
  files: [audioSample],  // MP3/WAV file
  description: 'Custom voice for support'
});

// Use in Vora
voice: {
  provider: 'elevenlabs',
  voiceId: voice.voice_id
}
```

### Professional Voice Clone

For higher quality clones:
1. Contact ElevenLabs for Professional tier
2. Provide 30+ minutes of audio
3. Receive high-fidelity voice clone

### Cloning Requirements

- Clear audio, minimal background noise
- Single speaker
- 1-5 minutes for instant clone
- 30+ minutes for professional clone

## Troubleshooting

<AccordionGroup>
  <Accordion title="Voice Sounds Robotic">
    If voice lacks naturalness:

    - Lower stability (try 0.3-0.4)
    - Increase similarity boost
    - Try a different voice
    - Check if model matches language
  </Accordion>

  <Accordion title="High Latency">
    If responses are slow:

    - Use `eleven_turbo_v2` model
    - Set `style: 0`
    - Enable streaming
    - Check network latency
  </Accordion>

  <Accordion title="Pronunciation Errors">
    If words are mispronounced:

    - Use phonetic alternatives
    - Try SSML markup
    - Test with different voices
    - Report to ElevenLabs for fixes
  </Accordion>
</AccordionGroup>
