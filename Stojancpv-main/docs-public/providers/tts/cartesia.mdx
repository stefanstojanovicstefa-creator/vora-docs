# Cartesia

Cartesia offers the fastest TTS with ultra-low latency optimized for real-time voice AI.

---

## Overview

| Aspect | Details |
|--------|---------|
| Provider ID | `cartesia` |
| Default Model | `sonic` |
| Streaming | ✅ Native support |
| Latency | ~100-150ms |
| Languages | 10+ |
| Voice Cloning | ❌ Not supported |

---

## Available Models

| Model | Quality | Latency | Best For |
|-------|---------|---------|----------|
| `sonic` | Very Good | ~100ms | Ultra-fast voice AI |
| `sonic-multilingual` | Good | ~150ms | Non-English languages |

<Note>
**Recommended**: `sonic` for English voice AI — fastest TTS available.
</Note>

## Configuration

### Basic Setup

```typescript
const agent = await vora.agents.create({
  name: 'Fast Agent',
  systemPrompt: 'You are a helpful assistant.',
  voice: {
    provider: 'cartesia',
    voiceId: 'atlas'
  }
});
```

### Full Configuration

```typescript
const agent = await vora.agents.create({
  name: 'Custom Voice Agent',
  systemPrompt: 'You are a professional assistant.',
  voice: {
    provider: 'cartesia',
    voiceId: 'atlas',
    model: 'sonic',
    speed: 1.0,
    emotion: 'neutral',
    outputFormat: 'pcm_16000'
  }
});
```

## Voice Settings

### Speed Control

Adjust speaking rate:

```typescript
voice: {
  provider: 'cartesia',
  voiceId: 'atlas',
  speed: 1.0  // 0.5 - 2.0
}
```

| Value | Effect |
|-------|--------|
| 0.5 - 0.8 | Slower, more deliberate |
| 1.0 | Normal speed |
| 1.1 - 1.3 | Slightly faster |
| 1.5 - 2.0 | Fast, energetic |

### Emotion Control

Set voice emotion:

```typescript
voice: {
  provider: 'cartesia',
  voiceId: 'luna',
  emotion: 'friendly'
}
```

| Emotion | Effect |
|---------|--------|
| `neutral` | Default, balanced |
| `friendly` | Warm, approachable |
| `professional` | Clear, business-like |
| `enthusiastic` | Energetic, excited |
| `calm` | Relaxed, soothing |

## Output Formats

| Format | Description | Use Case |
|--------|-------------|----------|
| `pcm_16000` | 16kHz PCM | Phone/telephony |
| `pcm_24000` | 24kHz PCM | Web applications |
| `pcm_44100` | 44.1kHz PCM | High quality |
| `mp3_128` | 128kbps MP3 | General use |
| `opus` | Opus codec | Low bandwidth |

Configure format:

```typescript
voice: {
  provider: 'cartesia',
  outputFormat: 'pcm_24000'
}
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Voice Sounds Flat">
    If voice lacks emotion:

    - Set `emotion` parameter explicitly
    - Try different voice IDs
    - Consider ElevenLabs for more expressive needs
  </Accordion>

  <Accordion title="Wrong Language Accent">
    If accent doesn't match:

    - Use `sonic-multilingual` model
    - Set explicit language code
    - Try different voices for that language
  </Accordion>

  <Accordion title="Audio Quality Issues">
    If audio sounds distorted:

    - Check output format matches your use case
    - Use PCM for lowest latency
    - Verify network connectivity
  </Accordion>
</AccordionGroup>
