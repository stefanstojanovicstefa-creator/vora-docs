## Choosing a Provider

| Priority | Recommended Provider | Why |
|----------|---------------------|-----|
| **Lowest latency** | Cartesia (Sonic-3) | ~80ms first byte — best for real-time conversation |
| **Most natural voices** | ElevenLabs | Gold standard for naturalness and expression |
| **Widest language coverage** | Google Cloud TTS | 75+ languages, 380+ voices |
| **Voice cloning** | ElevenLabs | Clone any voice with a short sample |
| **Cost at scale** | Google Cloud TTS | Cheapest per-character pricing |

## Configuration

Configure TTS in the Command Center [Voice Tab](/advanced/command-center/voice-tab):
1. Select your TTS provider
2. Browse and preview available voices
3. Set speech speed
4. Optionally set a fallback voice in the [Advanced Tab](/advanced/command-center/advanced-tab)

### API Configuration

```typescript
// ElevenLabs
const agent = await vora.agents.create({
  voice: {
    provider: 'elevenlabs',
    voiceId: 'EXAVITQu4vr4xnSDxMaL',
    model: 'eleven_turbo_v2_5',
    stability: 0.5,
    speed: 1.0
  }
});

// Cartesia (ultra-low latency)
const agent = await vora.agents.create({
  voice: {
    provider: 'cartesia',
    voiceId: 'sonic-3',
    language: 'en'
  }
});
```
