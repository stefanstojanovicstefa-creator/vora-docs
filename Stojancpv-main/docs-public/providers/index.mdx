# Providers Overview

Vora integrates with **12 ASR**, **11 TTS**, and **15 LLM** providers. Mix and match to build the optimal voice pipeline for your agents.

---

<Tabs>
  <Tab title="Simple">
    ## How Providers Work

    Every voice AI conversation uses three types of providers:

    1. **Speech-to-Text (STT)** - Converts what the caller says into text
    2. **AI Model (LLM)** - Understands the text and generates a response
    3. **Text-to-Speech (TTS)** - Converts the response back to speech

    Vora picks the best providers by default - you don't need to configure anything. Just create your agent and go.

    ### Default Providers

    | Component | Default | Why |
    |-----------|---------|-----|
    | **STT** | Deepgram | Fast and accurate |
    | **LLM** | Gemini Flash | Quick responses, 100+ languages |
    | **TTS** | ElevenLabs | Natural-sounding voices |

    <Tip>
      The defaults work great for most agents. You only need to change providers if you have specific language or performance requirements.
    </Tip>

    ## Configuring Providers

    Configure per-agent in the [Command Center](/advanced/command-center):
    - **Ears Tab** → STT provider and language
    - **Brain Tab** → LLM provider and model
    - **Voice Tab** → TTS provider and voice
    - **Advanced Tab** → Fallback chains and circuit breakers

    ## Fallback Strategies

    Configure automatic failover in the [Advanced Tab](/advanced/command-center/advanced-tab):

    ```
    Primary: Deepgram → Fallback: Google STT → Fallback: Azure STT
    Primary: Gemini Flash → Fallback: GPT-4o-mini
    Primary: ElevenLabs → Fallback: Cartesia
    ```

    ## Bring Your Own Keys

    Use your own provider API keys for higher rate limits and direct billing. See [API Keys](/advanced/api-keys) for setup.
