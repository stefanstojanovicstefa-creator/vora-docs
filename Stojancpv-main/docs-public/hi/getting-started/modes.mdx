# Simple aur Advanced Mode

Vora ka dashboard aur documentation do modes offer karte hain taaki aap hamesha apni zaroorat ke hisaab se sahi level ki detail dekhein.

---

## Kaise kaam karta hai

<Tabs>
  <Tab title="Simple">
    **Simple Mode** default experience hai. Yeh business users ke liye designed hai jo voice agents banane aur manage karne chahte hain bina technical details ki chinta ke.

    Simple Mode mein aap dekhenge:
    - Simplified agent creation flow (Interview aur URL Import methods)
    - Sirf essential settings (voice, language, greeting)
    - Clean dashboard key metrics ke saath
    - Guided workflows helpful tips ke saath

    <Tip>
      Simple Mode perfect hai agar aap jaldi se ek agent shuru karna chahte hain. Aap baad mein kabhi bhi Advanced Mode mein switch kar sakte hain.
    </Tip>
  </Tab>
  <Tab title="Advanced">
    **Advanced Mode** Vora platform ki poori power unlock karta hai. Yeh developers aur power users ke liye designed hai jinhe granular control chahiye.

    Advanced Mode mein aapko Simple Mode ka sab kuch milta hai plus:
    - **Command Center** - 9-tab agent builder poori provider control ke saath
    - **Prompt Mode** - Direct system prompt editing
    - **Custom Functions** - Custom tool calls banayein aur attach karein
    - **Monitoring** - Real-time system health aur metrics
    - **API Keys** - API keys manage karein aur apne provider keys laayein
    - **Provider Selection** - STT, TTS aur LLM providers choose aur configure karein

    <Info>
      Advanced Mode sabhi configuration options dikhata hai. Kuch features jaise Custom Functions aur Monitoring sirf Advanced Mode mein visible hain.
    </Info>
  </Tab>
</Tabs>

---

## Mode switch karna

### Dashboard mein

Sidebar ke top-right corner mein mode toggle par click karein:
- **Simple** (sparkles icon) - Simplified interface
- **Advanced** (settings icon) - Full developer interface

Aapka preference automatically save hota hai aur sessions ke beech persist karta hai.

## Kaun sa mode use karein?

<CardGroup cols={2}>
  <Card title="Simple Mode" icon="sparkles">
    **Yeh choose karein agar:**
    - Aap jaldi se agents banana chahte hain
    - Custom provider settings ki zaroorat nahi hai
    - Clean, guided interface prefer karte hain
    - Business user hain ya developer nahi hain
  </Card>
  <Card title="Advanced Mode" icon="gear">
    **Yeh choose karein agar:**
    - Providers aur models par poora control chahiye
    - Custom functions banana chahte hain
    - System performance monitor karna hai
    - Developer hain jo integrations bana rahe hain
  </Card>
</CardGroup>
