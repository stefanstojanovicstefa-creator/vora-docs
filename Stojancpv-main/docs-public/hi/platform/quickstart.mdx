# Quick Start

<Tabs>
  <Tab title="Simple">
    Apna pehla Vora voice agent 3 easy steps mein shuru karein. Bina coding ke.

    ## Shuru karne se pehle

    Aapko chahiye:
    - Vora account ([free mein sign up karein](https://app.voicevora.com/signup))
    - Testing ke liye microphone
    - Lagbhag 5 minute

    ---

    ## Step 1: Agent banayein

    <Steps>
      <Step title="Agents par jaayein">
        Dashboard se, sidebar mein **Agents** par click karein, phir **Create Agent** par click karein.
      </Step>
      <Step title="Interview Method choose karein">
        Creation method ke roop mein **Interview** choose karein. Vora aapse aapke agent ke baare mein sawaal poochega aur aapke liye bana dega.
      </Step>
      <Step title="Voice aur Language choose karein">
        Library se ek voice choose karein aur apne agent ki language select karein. Vora 100+ languages support karta hai.
      </Step>
    </Steps>

    ---

    ## Step 2: Agent test karein

    **Test** button par click karein aur baat karna shuru karein. Kuch scenarios try karein.

    ---

    ## Step 3: Deploy karein

    **Deploy** par click karein, channel choose karein (phone number ya web widget), aur **Activate** karein.

  <Tab title="Advanced">
    Providers aur configuration par poore control ke saath apna pehla Vora voice agent shuru karein.

    ## Shuru karne se pehle

    Aapko chahiye:
    - Vora account ([free mein sign up karein](https://app.voicevora.com/signup))
    - Testing ke liye microphone
    - Lagbhag 5 minute
    - (Optional) Custom configuration ke liye apne provider API keys

    <Info>
      Sidebar mein toggle click karke **Advanced Mode** mein hona ensure karein. Yeh Command Center aur provider settings unlock karta hai.
    </Info>

    ---

    ## Step 1: Agent banayein

    Creation method choose karein: Interview (recommended), URL Import, Prompt Mode, ya Command Center.

    ## Step 2: Providers configure karein

    | Component | Default Provider | Alternatives |
    |-----------|-----------------|--------------|
    | **STT** | Deepgram | Google, AssemblyAI, Soniox |
    | **LLM** | Gemini Flash | GPT-4o, Claude, Groq |
    | **TTS** | ElevenLabs | Cartesia, Google, Azure |

    ## Step 3: Test aur Deploy karein

    Simulator mein test karein, phir phone number, web widget ya API se deploy karein.
