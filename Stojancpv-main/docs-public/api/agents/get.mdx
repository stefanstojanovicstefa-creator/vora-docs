# Get Agents

Retrieve information about a specific agent or list all agents in your organization.

## Get a Specific Agent

Retrieve detailed information about a single agent by ID.

```
GET https://api.voicevora.com/v1/agents/:id
```

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | The agent ID (e.g., `agent_abc123`) |

## List All Agents

Retrieve a paginated list of all agents in your organization.

```
GET https://api.voicevora.com/v1/agents
```

### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `limit` | integer | `20` | Number of agents per page (max: 100) |
| `cursor` | string | - | Pagination cursor from previous response |
| `status` | string | - | Filter by status: `draft`, `deploying`, `deployed`, `error` |
| `order` | string | `desc` | Sort order: `asc` or `desc` |
| `order_by` | string | `created_at` | Sort field: `created_at`, `updated_at`, `name` |
| `search` | string | - | Search by name (partial match) |

### Examples

#### List All Agents

<CodeGroup>
```javascript JavaScript
const { data: agents, pagination } = await vora.agents.list();

console.log(`Found ${pagination.total} agents`);
agents.forEach(agent => {
  console.log(`- ${agent.name} (${agent.status})`);
});
```

```python Python
result = vora.agents.list()

print(f"Found {result.pagination.total} agents")
for agent in result.data:
    print(f"- {agent.name} ({agent.status})")
```

```bash cURL
curl "https://api.voicevora.com/v1/agents" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</CodeGroup>

#### Paginate Through All Agents

<CodeGroup>
```javascript JavaScript
async function* getAllAgents() {
  let cursor = null;

  do {
    const response = await vora.agents.list({
      limit: 50,
      cursor
    });

    for (const agent of response.data) {
      yield agent;
    }

    cursor = response.pagination.next_cursor;
  } while (cursor);
}

// Process all agents
for await (const agent of getAllAgents()) {
  console.log(`Processing: ${agent.name}`);
}
```

```python Python
def get_all_agents():
    cursor = None

    while True:
        response = vora.agents.list(limit=50, cursor=cursor)

        for agent in response.data:
            yield agent

        cursor = response.pagination.next_cursor
        if not cursor:
            break

# Process all agents
for agent in get_all_agents():
    print(f"Processing: {agent.name}")
```
</CodeGroup>

## Response Fields

### Compact vs Full Response

When listing agents, a compact response is returned with essential fields only. Use `GET /agents/:id` for the full agent object including the complete `config` and `stats`.

| Endpoint | Config Fields | Stats Fields |
|----------|---------------|--------------|
| List (`GET /agents`) | `voice_id`, `llm_provider`, `llm_model` | `total_calls`, `avg_duration_seconds` |
| Get (`GET /agents/:id`) | All fields | All fields including `last_call_at` |
