# API Reference

The Vora API gives you programmatic access to create, configure, and manage voice AI agents. Build powerful voice experiences with our RESTful API.

<Info>
  **Base URL:** `https://api.voicevora.com/v1`

  All endpoints require authentication. See the [Authentication Guide](/api/authentication) to get started.
</Info>

## Getting Started

<Steps>
  <Step title="Get Your API Key">
    Sign up at [app.voicevora.com](https://app.voicevora.com) and generate an API key from Settings → API Keys.
  </Step>
  <Step title="Make Your First Request">
    Test your connection by listing your agents:
    ```bash
    curl https://api.voicevora.com/v1/agents \
      -H "Authorization: Bearer YOUR_API_KEY"
    ```
  </Step>
  <Step title="Create an Agent">
    Create your first voice agent with a simple POST request. See the [Agents API](/api/agents) for details.
  </Step>
</Steps>

## Quick Reference

### HTTP Methods

| Method | Description |
|--------|-------------|
| `GET` | Retrieve resources |
| `POST` | Create new resources |
| `PUT` | Update existing resources |
| `DELETE` | Remove resources |

### Common Headers

| Header | Value | Description |
|--------|-------|-------------|
| `Authorization` | `Bearer YOUR_API_KEY` | Required for authentication |
| `Content-Type` | `application/json` | Required for POST/PUT requests |
| `Accept` | `application/json` | Response format |

### Response Format

All responses follow a consistent JSON structure:

```json
// Success response
{
  "data": { ... },
  "meta": {
    "request_id": "req_abc123"
  }
}

// List response
{
  "data": [ ... ],
  "pagination": {
    "has_more": true,
    "next_cursor": "xyz789",
    "total": 47
  }
}

// Error response
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Invalid parameters",
    "details": { ... }
  }
}
```
