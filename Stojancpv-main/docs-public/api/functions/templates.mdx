# Function Templates

Copy and customize these pre-built function templates for common integration scenarios.

## E-Commerce

### Order Status Lookup

```javascript
const orderStatus = await vora.functions.create({
  name: 'get_order_status',
  description: 'Look up order status, shipping information, and estimated delivery by order ID',
  parameters: {
    type: 'object',
    properties: {
      order_id: {
        type: 'string',
        description: 'Order ID or tracking number (e.g., ORD-12345)'
      }
    },
    required: ['order_id']
  },
  endpoint: {
    url: 'https://your-api.com/orders/{{order_id}}',
    method: 'GET',
    headers: {
      'Authorization': 'Bearer {{ORDER_API_KEY}}'
    }
  },
  response_template: `Order {{order_id}} is {{status}}.
{{#if tracking_number}}Tracking: {{tracking_number}}.{{/if}}
{{#if estimated_delivery}}Expected delivery: {{estimated_delivery}}.{{/if}}`,
  error_response: "I couldn't find that order. Please verify the order ID.",
  timeout_ms: 5000
});
```

### Return / Refund Request

```javascript
const createReturn = await vora.functions.create({
  name: 'create_return',
  description: 'Initiate a return or refund request for an order',
  parameters: {
    type: 'object',
    properties: {
      order_id: {
        type: 'string',
        description: 'Order ID to return'
      },
      reason: {
        type: 'string',
        enum: ['damaged', 'wrong_item', 'not_as_described', 'changed_mind', 'defective'],
        description: 'Reason for return'
      },
      items: {
        type: 'array',
        items: { type: 'string' },
        description: 'Specific item IDs to return (optional, all items if not specified)'
      }
    },
    required: ['order_id', 'reason']
  },
  endpoint: {
    url: 'https://your-api.com/returns',
    method: 'POST',
    headers: {
      'Authorization': 'Bearer {{ORDER_API_KEY}}',
      'Content-Type': 'application/json'
    },
    body_template: '{"order_id":"{{order_id}}","reason":"{{reason}}","items":{{items}}}'
  },
  response_template: `Your return request is confirmed. Return ID: {{return_id}}.
Print your return label at {{label_url}}.
Refund will process within 5-7 business days after we receive the items.`,
  timeout_ms: 8000
});
```

## Scheduling & Appointments

### Book Appointment

```javascript
const bookAppointment = await vora.functions.create({
  name: 'book_appointment',
  description: 'Schedule an appointment for a specific date, time, and service type',
  parameters: {
    type: 'object',
    properties: {
      service: {
        type: 'string',
        enum: ['consultation', 'follow-up', 'new-patient', 'procedure'],
        description: 'Type of appointment'
      },
      date: {
        type: 'string',
        format: 'date',
        description: 'Preferred date (YYYY-MM-DD)'
      },
      time: {
        type: 'string',
        description: 'Preferred time (HH:MM)'
      },
      patient_name: {
        type: 'string',
        description: 'Patient full name'
      },
      phone: {
        type: 'string',
        description: 'Contact phone number'
      },
      notes: {
        type: 'string',
        description: 'Special requests or notes'
      }
    },
    required: ['service', 'date', 'time', 'patient_name', 'phone']
  },
  endpoint: {
    url: 'https://your-api.com/appointments',
    method: 'POST',
    headers: {
      'Authorization': 'Bearer {{SCHEDULING_API_KEY}}',
      'Content-Type': 'application/json'
    },
    body_template: '{"service":"{{service}}","date":"{{date}}","time":"{{time}}","patient":{"name":"{{patient_name}}","phone":"{{phone}}"},"notes":"{{notes}}"}'
  },
  response_template: `Your {{service}} appointment is confirmed for {{date}} at {{time}}.
Confirmation number: {{confirmation_id}}.
We'll send a reminder to {{phone}} the day before.`,
  error_response: "That time slot isn't available. Would you like to try a different time?",
  timeout_ms: 10000
});
```

## Weather & Location

### Current Weather

```javascript
const getWeather = await vora.functions.create({
  name: 'get_weather',
  description: 'Get current weather conditions for a city or location',
  parameters: {
    type: 'object',
    properties: {
      location: {
        type: 'string',
        description: 'City name, zip code, or coordinates'
      },
      units: {
        type: 'string',
        enum: ['celsius', 'fahrenheit'],
        default: 'fahrenheit',
        description: 'Temperature units'
      }
    },
    required: ['location']
  },
  endpoint: {
    url: 'https://api.weatherapi.com/v1/current.json',
    method: 'GET',

    query_template: {
      'q': '{{location}}'
    }
  },
  response_template: `In {{location.name}}, it's currently {{current.temp_f}}°F with {{current.condition.text}}.
Humidity: {{current.humidity}}%.

});
```

### Weather Forecast

```javascript
const getForecast = await vora.functions.create({
  name: 'get_weather_forecast',
  description: 'Get weather forecast for the next few days',
  parameters: {
    type: 'object',
    properties: {
      location: {
        type: 'string',
        description: 'City name or zip code'
      },
      days: {
        type: 'integer',
        default: 3,
        minimum: 1,
        maximum: 7,
        description: 'Number of days to forecast'
      }
    },
    required: ['location']
  },
  endpoint: {
    url: 'https://api.weatherapi.com/v1/forecast.json',
    method: 'GET',
    headers: {
      'X-API-Key': '{{WEATHER_API_KEY}}'
    },
    query_template: {
      'q': '{{location}}',
      'days': '{{days}}'
    }
  },
  response_template: `Forecast for {{location.name}}:
{{#each forecast.forecastday}}
{{date}}: High {{day.maxtemp_f}}°, Low {{day.mintemp_f}}°. {{day.condition.text}}.
{{/each}}`,
  timeout_ms: 3000
});
```

## Using Templates

### 1. Copy the Template

Select a template that matches your use case.

### 2. Customize the Endpoint

Replace placeholder URLs and add your API details:

```javascript
endpoint: {
  url: 'https://YOUR-ACTUAL-API.com/endpoint',  // Your API URL
  headers: {
    'Authorization': 'Bearer {{YOUR_SECRET_NAME}}'  // Reference your secret
  }
}
```

### 3. Store Required Secrets

```javascript
await vora.secrets.set('YOUR_SECRET_NAME', 'your-actual-api-key');
```

### 4. Test the Function

```javascript
const result = await vora.functions.test(fn.id, {
  // Sample parameters
});
console.log(result);
```

### 5. Attach to Agent

```javascript
await vora.agents.update('agent_id', {
  functions: [fn.id]
});
```
