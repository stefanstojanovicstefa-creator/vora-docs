# Mode Sederhana & Lanjutan

Dashboard dan dokumentasi Vora menawarkan dua mode sehingga Anda selalu melihat tingkat detail yang tepat.

---

## Cara Kerjanya

<Tabs>
  <Tab title="Sederhana">
    **Mode Sederhana** adalah pengalaman default. Dirancang untuk pengguna bisnis yang ingin membuat dan mengelola agen suara tanpa khawatir tentang detail teknis.

    Dalam Mode Sederhana, Anda akan melihat:
    - Alur pembuatan agen yang disederhanakan (metode Wawancara dan Impor URL)
    - Hanya pengaturan penting (suara, bahasa, sapaan)
    - Dashboard bersih dengan metrik utama
    - Alur kerja terpandu dengan tips berguna

    <Tip>
      Mode Sederhana sempurna jika Anda ingin menjalankan agen dengan cepat. Anda selalu bisa beralih ke Mode Lanjutan nanti.
    </Tip>
  </Tab>
  <Tab title="Lanjutan">
    **Mode Lanjutan** membuka kekuatan penuh platform Vora. Dirancang untuk pengembang dan pengguna mahir yang membutuhkan kontrol detail.

    Dalam Mode Lanjutan Anda mendapat semua yang ada di Mode Sederhana plus:
    - **Pusat Komando** - Pembuat agen 9 tab dengan kontrol penyedia penuh
    - **Mode Prompt** - Pengeditan prompt sistem langsung
    - **Fungsi Kustom** - Buat dan lampirkan panggilan alat kustom
    - **Pemantauan** - Kesehatan sistem real-time dan metrik
    - **Kunci API** - Kelola kunci API dan bawa kunci penyedia Anda sendiri

    <Info>
      Mode Lanjutan menampilkan semua opsi konfigurasi. Beberapa fitur seperti Fungsi Kustom dan Pemantauan hanya terlihat di Mode Lanjutan.
    </Info>
  </Tab>
</Tabs>

---

## Beralih Mode

### Di Dashboard

Klik toggle mode di sudut kanan atas sidebar:
- **Sederhana** (ikon bintang) - Antarmuka disederhanakan
- **Lanjutan** (ikon pengaturan) - Antarmuka pengembang lengkap

Preferensi Anda disimpan otomatis dan bertahan di seluruh sesi.

## Mode Mana yang Harus Digunakan?

<CardGroup cols={2}>
  <Card title="Mode Sederhana" icon="sparkles">
    **Pilih ini jika:**
    - Ingin membuat agen dengan cepat
    - Tidak memerlukan pengaturan penyedia kustom
    - Lebih suka antarmuka bersih dan terpandu
    - Pengguna bisnis atau bukan pengembang
  </Card>
  <Card title="Mode Lanjutan" icon="gear">
    **Pilih ini jika:**
    - Memerlukan kontrol penuh atas penyedia dan model
    - Ingin membangun fungsi kustom
    - Perlu memantau kinerja sistem
    - Pengembang yang membangun integrasi
  </Card>
</CardGroup>
