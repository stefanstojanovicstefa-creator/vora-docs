## Vora kyun?

<AccordionGroup>
  <Accordion title="Voice-first design">
    Real-time voice conversations ke liye ground up se banaya gaya. 500ms se kam latency, natural turn-taking, aur barge-in support LiveKit WebRTC ke zariye.
  </Accordion>
  <Accordion title="100+ bhashaayein supported">
    Arabic aur Hindi se lekar Vietnamese aur Thai tak. Vora 100+ bhashaayein support karta hai 12 ASR, 11 TTS aur 15 LLM providers ke zariye.
  </Accordion>
  <Accordion title="Apne providers laayein">
    LLM ke liye Gemini, GPT-4, Claude ya Groq use karein. STT ke liye Deepgram, Google ya AssemblyAI. TTS ke liye ElevenLabs, Cartesia ya Google.
  </Accordion>
  <Accordion title="Simple aur Advanced Mode">
    Business users ko Simple Mode mein streamlined experience milta hai. Developers Advanced Mode mein Command Center, custom functions, monitoring aur API access unlock karte hain.
  </Accordion>
</AccordionGroup>

## Shuru karein

<Steps>
  <Step title="Account banayein">
    [app.voicevora.com](https://app.voicevora.com) par sign up karein - credit card ke bina free tier available hai.
  </Step>
  <Step title="Apna pehla agent banayein">
    **Interview** method use karein - bas describe karein aap kya chahte hain aur Vora aapke liye bana dega.
  </Step>
  <Step title="Test aur deploy karein">
    Simulator mein test karein, phir phone number ya web widget se deploy karein.
  </Step>
</Steps>
