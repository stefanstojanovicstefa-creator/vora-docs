    ### Agent ki bhasha kaise set karein

    <Steps>
      <Step title="Apna agent kholein">
        Agents par jaayein aur apna agent select karein.
      </Step>
      <Step title="Bhasha choose karein">
        Voice settings mein, dropdown se bhasha select karein.
      </Step>
      <Step title="Save karein">
        Save par click karein. Aapka agent ab selected bhasha mein samajhta aur bolta hai.
      </Step>
    </Steps>

    ## Provider Coverage Matrix

    Vora **12 ASR**, **11 TTS** aur **15 LLM** providers ke saath integrate hota hai 100+ languages par.

    Detailed comparison ke liye:
    - [ASR Providers](/languages/asr-providers) - 12 speech-to-text providers ki tulna
    - [TTS Providers](/languages/tts-providers) - 11 text-to-speech providers ki tulna

    ### Configuration

    Command Center mein har agent ke liye bhasha set karein:
    - **Ears Tab** - Speech recognition ke liye STT language
    - **Voice Tab** - Speech output ke liye TTS language
    - **Brain Tab** - Response generation ke liye LLM language context
