# Mulai Cepat

<Tabs>
  <Tab title="Sederhana">
    Jalankan agen suara Vora pertama Anda dalam 3 langkah mudah. Tanpa coding.

    ## Sebelum Mulai

    Anda memerlukan:
    - Akun Vora ([daftar gratis](https://app.voicevora.com/signup))
    - Mikrofon untuk pengujian
    - Sekitar 5 menit

    ---

    ## Langkah 1: Buat Agen Anda

    <Steps>
      <Step title="Buka Agen">
        Dari dashboard, klik **Agen** di sidebar, lalu klik **Buat Agen**.
      </Step>
      <Step title="Pilih Metode Wawancara">
        Pilih **Wawancara** sebagai metode pembuatan. Vora akan mengajukan pertanyaan dan membuatkan agen untuk Anda.
      </Step>
      <Step title="Pilih Suara dan Bahasa">
        Pilih suara dari perpustakaan dan pilih bahasa agen Anda. Vora mendukung 100+ bahasa.
      </Step>
    </Steps>

    ---

    ## Langkah 2: Uji Agen Anda

    Klik tombol **Uji** dan mulai berbicara. Coba beberapa skenario.

    ---

    ## Langkah 3: Deploy

    Klik **Deploy**, pilih saluran (nomor telepon atau widget web), dan klik **Aktifkan**.
  </Tab>
  <Tab title="Lanjutan">
    Jalankan agen suara Vora pertama dengan kontrol penuh atas penyedia dan konfigurasi.

    ## Sebelum Mulai

    Anda memerlukan:
    - Akun Vora ([daftar gratis](https://app.voicevora.com/signup))
    - Mikrofon untuk pengujian
    - Sekitar 5 menit
    - (Opsional) Kunci API penyedia sendiri

    ---

    ## Langkah 1: Buat Agen

    Pilih metode pembuatan: Wawancara (direkomendasikan), Impor URL, Mode Prompt, atau Pusat Komando.

    ## Langkah 2: Konfigurasi Penyedia

    | Komponen | Penyedia Default | Alternatif |
    |----------|-----------------|------------|
    | **STT** | Deepgram | Google, AssemblyAI, Soniox |
    | **LLM** | Gemini Flash | GPT-4o, Claude, Groq |
    | **TTS** | ElevenLabs | Cartesia, Google, Azure |

    ## Langkah 3: Uji dan Deploy

    Uji di simulator, lalu deploy via nomor telepon, widget web, atau API.
