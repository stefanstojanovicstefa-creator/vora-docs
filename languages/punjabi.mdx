# Punjabi

<Tabs>
  <Tab title="Simple">
    Vora fully supports **Punjabi** for voice AI agents. ~125M speakers in India (Punjab) and Pakistan.

    ### Setting Up a Punjabi Agent

    <Steps>
      <Step title="Create your agent">
        Use any creation method (Interview, URL Import, etc.)
      </Step>
      <Step title="Set language to Punjabi">
        In voice settings, select **Punjabi** from the language dropdown.
      </Step>
      <Step title="Choose a voice">
        Select from available Punjabi voices in the voice library.
      </Step>
      <Step title="Test and deploy">
        Test with the simulator, then deploy as usual.
      </Step>
    </Steps>

    <Tip>
      For best results with Punjabi, we recommend Sarvam AI voices for the most natural speech.
    </Tip>
  </Tab>
  <Tab title="Advanced">
    ## Provider Matrix

    | Detail | Value |
    |--------|-------|
    | **Code** | pa-IN |
    | **Region** | India (Punjab) and Pakistan |
    | **Speakers** | ~125M |
    | **Best ASR** | Sarvam AI |
    | **Best TTS** | Sarvam AI |
    | **Best LLM** | Sarvam 2B |

    ### Available Providers

    | Type | Providers |
    |------|-----------|
    | **ASR** | Sarvam AI (Best), Google, Azure, AWS, Deepgram, Soniox |
    | **TTS** | Sarvam AI (Best), Google, Azure, Murf AI |
    | **LLM** | Sarvam 2B (Best), GPT-4o, Gemini, Claude |

    ### Recommended Stack

    For the best Punjabi voice experience:
    - **STT**: Sarvam AI (best accuracy for Punjabi)
    - **LLM**: Sarvam 2B (best Punjabi understanding)
    - **TTS**: Sarvam AI (most natural Punjabi speech)

    ### Configuration

    In the Command Center:
    1. **Ears Tab** → Set STT provider to Sarvam AI, language to `pa-IN`
    2. **Brain Tab** → Set LLM to Sarvam 2B
    3. **Voice Tab** → Set TTS to Sarvam AI, select a Punjabi voice
  </Tab>
