# Hindi

<Tabs>
  <Tab title="Simple">
    Vora fully supports **Hindi** for voice AI agents. ~600M speakers in India.

    ### Setting Up a Hindi Agent

    <Steps>
      <Step title="Create your agent">
        Use any creation method (Interview, URL Import, etc.)
      </Step>
      <Step title="Set language to Hindi">
        In voice settings, select **Hindi** from the language dropdown.
      </Step>
      <Step title="Choose a voice">
        Select from available Hindi voices in the voice library.
      </Step>
      <Step title="Test and deploy">
        Test with the simulator, then deploy as usual.
      </Step>
    </Steps>

    <Tip>
      For best results with Hindi, we recommend Sarvam AI voices for the most natural speech.
    </Tip>
  </Tab>
  <Tab title="Advanced">
    ## Provider Matrix

    | Detail | Value |
    |--------|-------|
    | **Code** | hi-IN |
    | **Region** | India |
    | **Speakers** | ~600M |
    | **Best ASR** | Sarvam AI |
    | **Best TTS** | Sarvam AI |
    | **Best LLM** | Sarvam 2B |

    ### Available Providers

    | Type | Providers |
    |------|-----------|
    | **ASR** | Sarvam AI (Best), Google, Azure, AWS, Deepgram, Soniox, AssemblyAI, ElevenLabs, Speechmatics, iFLYTEK |
    | **TTS** | Sarvam AI (Best), Google, Azure, AWS Polly, ElevenLabs, Murf AI, Speechify, Inworld |
    | **LLM** | Sarvam 2B (Best), GPT-4o, Gemini, Claude, Cohere Aya |

    ### Recommended Stack

    For the best Hindi voice experience:
    - **STT**: Sarvam AI (best accuracy for Hindi)
    - **LLM**: Sarvam 2B (best Hindi understanding)
    - **TTS**: Sarvam AI (most natural Hindi speech)

    ### Configuration

    In the Command Center:
    1. **Ears Tab** → Set STT provider to Sarvam AI, language to `hi-IN`
    2. **Brain Tab** → Set LLM to Sarvam 2B
    3. **Voice Tab** → Set TTS to Sarvam AI, select a Hindi voice
  </Tab>
