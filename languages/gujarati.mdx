# Gujarati

<Tabs>
  <Tab title="Simple">
    Vora fully supports **Gujarati** for voice AI agents. ~56M speakers in India (Gujarat).

    ### Setting Up a Gujarati Agent

    <Steps>
      <Step title="Create your agent">
        Use any creation method (Interview, URL Import, etc.)
      </Step>
      <Step title="Set language to Gujarati">
        In voice settings, select **Gujarati** from the language dropdown.
      </Step>
      <Step title="Choose a voice">
        Select from available Gujarati voices in the voice library.
      </Step>
      <Step title="Test and deploy">
        Test with the simulator, then deploy as usual.
      </Step>
    </Steps>

    <Tip>
      For best results with Gujarati, we recommend Sarvam AI voices for the most natural speech.
    </Tip>
  </Tab>
  <Tab title="Advanced">
    ## Provider Matrix

    | Detail | Value |
    |--------|-------|
    | **Code** | gu-IN |
    | **Region** | India (Gujarat) |
    | **Speakers** | ~56M |
    | **Best ASR** | Sarvam AI |
    | **Best TTS** | Sarvam AI |
    | **Best LLM** | Sarvam 2B |

    ### Available Providers

    | Type | Providers |
    |------|-----------|
    | **ASR** | Sarvam AI (Best), Google, Azure, AWS, Deepgram, Soniox |
    | **TTS** | Sarvam AI (Best), Google, Azure, Murf AI, Speechify |
    | **LLM** | Sarvam 2B (Best), GPT-4o, Gemini, Claude |

    ### Recommended Stack

    For the best Gujarati voice experience:
    - **STT**: Sarvam AI (best accuracy for Gujarati)
    - **LLM**: Sarvam 2B (best Gujarati understanding)
    - **TTS**: Sarvam AI (most natural Gujarati speech)

    ### Configuration

    In the Command Center:
    1. **Ears Tab** → Set STT provider to Sarvam AI, language to `gu-IN`
    2. **Brain Tab** → Set LLM to Sarvam 2B
    3. **Voice Tab** → Set TTS to Sarvam AI, select a Gujarati voice
  </Tab>
