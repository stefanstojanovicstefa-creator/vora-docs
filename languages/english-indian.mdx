# English (Indian)

<Tabs>
  <Tab title="Simple">
    Vora fully supports **English (Indian)** for voice AI agents. ~125M speakers in India.

    ### Setting Up an English (Indian) Agent

    <Steps>
      <Step title="Create your agent">
        Use any creation method (Interview, URL Import, etc.)
      </Step>
      <Step title="Set language to English (Indian)">
        In voice settings, select **English (Indian)** from the language dropdown.
      </Step>
      <Step title="Choose a voice">
        Select from available English (Indian) voices in the voice library.
      </Step>
      <Step title="Test and deploy">
        Test with the simulator, then deploy as usual.
      </Step>
    </Steps>

    <Tip>
      For best results with English (Indian), we recommend ElevenLabs voices for the most natural speech.
    </Tip>
  </Tab>
  <Tab title="Advanced">
    ## Provider Matrix

    | Detail | Value |
    |--------|-------|
    | **Code** | en-IN |
    | **Region** | India |
    | **Speakers** | ~125M |
    | **Best ASR** | Deepgram |
    | **Best TTS** | ElevenLabs |
    | **Best LLM** | GPT-4o |

    ### Available Providers

    | Type | Providers |
    |------|-----------|
    | **ASR** | Sarvam AI, Google, Azure, AWS, Deepgram, Soniox, AssemblyAI, ElevenLabs, Speechmatics |
    | **TTS** | Sarvam AI, Google, Azure, AWS Polly, ElevenLabs, Murf AI, Speechify |
    | **LLM** | GPT-4o, Gemini, Claude, Sarvam 2B |

    ### Recommended Stack

    For the best English (Indian) voice experience:
    - **STT**: Deepgram (best accuracy for English (Indian))
    - **LLM**: GPT-4o (best English (Indian) understanding)
    - **TTS**: ElevenLabs (most natural English (Indian) speech)

    ### Configuration

    In the Command Center:
    1. **Ears Tab** → Set STT provider to Deepgram, language to `en-IN`
    2. **Brain Tab** → Set LLM to GPT-4o
    3. **Voice Tab** → Set TTS to ElevenLabs, select an English (Indian) voice
  </Tab>
