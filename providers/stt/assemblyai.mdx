## Configuration

### Basic Setup

```typescript
const agent = await vora.agents.create({
  name: 'Accurate Agent',
  systemPrompt: 'You are a helpful assistant.',
  stt: {
    provider: 'assemblyai',
    model: 'best'
  }
});
```

### Full Configuration

```typescript
const agent = await vora.agents.create({
  name: 'Custom Agent',
  systemPrompt: 'You are a specialized assistant.',
  stt: {
    provider: 'assemblyai',
    model: 'best',
    languageCode: 'en_us',
    punctuate: true,
    formatText: true,
    wordBoost: ['product-name', 'technical-term'],
    filterProfanity: false,
    redactPii: false,
    speakerLabels: false
  }
});
```

## Advanced Features

### Speaker Diarization

Identify speakers in multi-party conversations:

```typescript
stt: {
  provider: 'assemblyai',
  speakerLabels: true,
  speakersExpected: 2  // Optional hint
}

// Result:
// Speaker A: "Hello, how can I help?"
// Speaker B: "I have a question about..."
```

### PII Redaction

Automatically redact sensitive information:

```typescript
stt: {
  provider: 'assemblyai',
  redactPii: true,
  redactPiiPolicies: [
    'us_social_security_number',
    'credit_card_number',
    'phone_number',
    'email_address'
  ],
  redactPiiSub: 'hash'  // or 'entity_name'
}

// Result:
// "My SSN is [SSN_HASH_12345]"
```

### Content Safety

Detect sensitive content:

```typescript
stt: {
  provider: 'assemblyai',
  contentSafety: true
}

// Returns flags for:
// - Hate speech
// - Violence
// - Profanity
// - Sensitive topics
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Accuracy Issues">
    If transcription quality is low:

    - Use `best` model instead of `nano`
    - Add word boosts for domain terms
    - Check audio quality
    - Verify correct language code
  </Accordion>

  <Accordion title="Latency Concerns">
    If responses are too slow:

    - Switch to `nano` model
    - Lower silence threshold
    - Check network latency
    - Consider Deepgram for ultra-low latency
  </Accordion>

  <Accordion title="Missing Words">
    If words are dropped:

    - Add to word boost list
    - Increase boost strength
    - Check audio clarity
    - Ensure proper pronunciation in prompts
  </Accordion>
</AccordionGroup>
