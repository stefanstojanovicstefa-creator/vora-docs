## Configuration

### Basic Setup

```typescript
const agent = await vora.agents.create({
  name: 'Multilingual Agent',
  systemPrompt: 'You are a helpful assistant.',
  stt: {
    provider: 'google',
    model: 'latest_long'
  }
});
```

### Full Configuration

```typescript
const agent = await vora.agents.create({
  name: 'Custom Agent',
  systemPrompt: 'You are a specialized assistant.',
  stt: {
    provider: 'google',
    model: 'latest_long',
    languageCode: 'en-US',
    alternativeLanguageCodes: ['es-ES', 'fr-FR'],
    enableAutomaticPunctuation: true,
    enableWordTimeOffsets: false,
    profanityFilter: false,
    speechContexts: [
      {
        phrases: ['product-name', 'technical-term'],
        boost: 10
      }
    ]
  }
});
```

## Speech Contexts

Boost recognition of specific phrases:

```typescript
stt: {
  provider: 'google',
  speechContexts: [
    {
      phrases: [
        'Vora',
        'LiveKit',
        '$OOV_CLASS_ALPHANUMERIC_SEQUENCE'  // Order IDs, etc.
      ],
      boost: 20  // -100 to 100
    }
  ]
}
```

### Class Tokens

Use built-in classes:

| Token | Matches |
|-------|---------|
| `$OOV_CLASS_ALPHANUMERIC_SEQUENCE` | Order123, ID456 |
| `$OOV_CLASS_DIGIT_SEQUENCE` | 123456 |
| `$OPERAND` | Numbers |

## API Key Setup

### Service Account

Google STT requires a service account:

```bash
# Environment variable pointing to credentials file
GOOGLE_APPLICATION_CREDENTIALS=/path/to/service-account.json
```

### Get Credentials

1. Go to [Google Cloud Console](https://console.cloud.google.com)
2. Enable Speech-to-Text API
3. Create a service account
4. Download JSON key file
5. Set environment variable

### Alternative: API Key

For simpler setup (less secure):

```bash
GOOGLE_API_KEY=AIza...
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Authentication Errors">
    If authentication fails:

    - Verify service account JSON exists
    - Check GOOGLE_APPLICATION_CREDENTIALS path
    - Ensure Speech-to-Text API is enabled
    - Verify project billing is active
  </Accordion>

  <Accordion title="Language Detection Issues">
    If wrong language is detected:

    - Set explicit languageCode
    - Limit alternativeLanguageCodes
    - Consider language-specific agents
  </Accordion>

  <Accordion title="Accuracy Problems">
    If transcription quality is low:

    - Use speechContexts for domain terms
    - Try enhanced model for phone audio
    - Check audio sample rate (16kHz ideal)
    - Reduce background noise
  </Accordion>
</AccordionGroup>
