## Choosing a Provider

### By Priority

| Priority | Recommended Provider | Why |
|----------|---------------------|-----|
| **Most natural English** | ElevenLabs | Gold standard for naturalness and expression |
| **Lowest latency** | Cartesia Sonic (~50ms), Murf AI (~80ms) | Ultra-fast first byte |
| **Widest language coverage** | Google Cloud TTS | 75+ languages, 380+ voices |
| **Indian languages** | Sarvam AI Bulbul | Purpose-built for 11 Indian languages |
| **Arabic** | Munsit (CNTXT) | Best Arabic dialect voices |
| **Lowest cost** | Murf AI Falcon ($0.01/min) | 10x cheaper than ElevenLabs |
| **Voice cloning** | ElevenLabs, Speechify | Clone any voice |
| **Enterprise** | Azure Neural TTS | Custom voice models, compliance |

## Configuration

Configure TTS providers in the Command Center [Voice Tab](/advanced/command-center/voice-tab):
1. Select your TTS provider
2. Browse and preview available voices
3. Set speech speed
4. Configure a fallback voice in the [Advanced Tab](/advanced/command-center/advanced-tab)

### API Configuration

```typescript
const agent = await vora.agents.create({
  name: 'Voice Agent',
  systemPrompt: 'You are a helpful assistant.',
  voice: {
    provider: 'elevenlabs',
    voiceId: 'EXAVITQu4vr4xnSDxMaL',
    model: 'eleven_turbo_v2_5',
    stability: 0.5,
    similarityBoost: 0.75,
    speed: 1.0
  }
});
```
